A propos :

Ceci est l'utilaire d'utilisation de l'API Restful de Pecudep Pecude est un logiciel de gestion des petites dépenses journalière

Services :

Read me: http://localhost:8080/com.pecudep/

1. Status : http://localhost:8080/com.pecudep/api/v1/status

2. Version : http://localhost:8080/com.pecudep/api/v1/status/version

Dépense :

3. Ajouter une dépense : http://localhost:8080/com.pecudep/api/depenses/postDepense/{id}

4. Afficher une dépense : http://localhost:8080/com.pecudep/api/depenses/getDepense/{id}

5. Modifier une dépense : http://localhost:8080/com.pecudep/api/depenses/putDepense/{id}

6. Supprimer une dépense : http://localhost:8080/com.pecudep/api/depenses/deleteDepense/{id}

7. Lister les dépenses par utilisateur et dossier : http://localhost:8080/com.pecudep/api/depenses/{utilisateur}/{dossier}

8. Le bilan d'un utilisateur jusqu'à la date actuelle : http://localhost:8080/com.pecudep/api/depenses/getBilan/{utilisateur}
